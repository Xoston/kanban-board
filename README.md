## Описание

Минималистичное приложение для управления задачами с тремя колонками:
- **Сделать** - новые задачи
- **В процессе** - текущие задачи  
- **Сделано** - завершенные задачи

## Возможности

- Добавление новых задач
- Перемещение между колонками
- Удаление задач
- Адаптивный дизайн
